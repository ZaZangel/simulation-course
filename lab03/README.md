# Клеточные автоматы. Моделирование лесных пожаров (GUI)

## 1. ЦЕЛЬ РАБОТЫ
Реализовать моделирование возникновения и распространения лесных пожаров с использованием двумерного клеточного автомата с графическим интерфейсом.

## 2. ТЕОРЕТИЧЕСКИЕ СВЕДЕНИЯ

**Что такое клеточный автомат?**  
Клеточный автомат — это дискретная модель, состоящая из сетки клеток, каждая из которых находится в одном из конечного числа состояний. Состояние каждой клетки меняется по определённым правилам в зависимости от состояний соседних клеток.

**Двумерный клеточный автомат** — это сетка размером N×M (в нашем случае 50×50), где:
- Каждая клетка имеет координаты (x, y)
- Клетка может быть в одном из состояний
- Обновление происходит одновременно для всех клеток на каждом шаге

**Состояния клеток в моей модели:**

| Состояние | Цвет | Описание |
| :--- | :--- | :--- |
| 🌲 Tree (Дерево) | Зелёный | Здоровое дерево |
| 🔥 Fire (Огонь) | Оранжево-красный | Горящее дерево |
| ⬛ Empty (Пусто) | Чёрный | Пустая клетка (сгорела или нет дерева) |
| 🌊 Barrier (Река) | Голубой | Преграда (не горит) |

**Правила модели:**
- **Базовое правило:** Дерево загорается от соседнего огня с вероятностью 75%
- **Ветер:** Увеличивает вероятность распространения огня в направлении ветра на 20%
- **Молнии:** Случайно поджигают деревья с вероятностью 0.3% на каждом шаге
- **Река:** Создаёт непреодолимую преграду для огня
- **Влажность:** Снижает вероятность возгорания (чем выше влажность, тем меньше горит)
- **Рост:** На пустых клетках с вероятностью 1% вырастает новое дерево

## 3. ЗАПУСК ПРОГРАММЫ

**Требования:**
- Visual Studio 2019 или выше
- .NET Framework 4.7.2 или выше
- Язык C# 8.0+

**Инструкция по запуску:**
1. Открыть проект в Visual Studio
2. Убедиться, что в Program.cs стоит `[STAThread]` перед `Main()`
3. Собрать проект: **Build → Build Solution** (F6)
4. Запустить: **Debug → Start Debugging** (F5)

**Управление программой:**

| Кнопка | Действие |
| :--- | :--- |
| ▶ Старт | Запустить симуляцию |
| ⏹ Стоп | Остановить симуляцию |
| 🔄 Сброс | Пересоздать лес заново |
| 🔥 Поджечь (центр) | Создать очаг пожара в центре |
| 🎲 Случайные очаги | Создать 3 случайных очага |
| ✅ Все правила | Включить все эффекты сразу |
| ⭕ Без правил | Выключить все эффекты |

**Чекбоксы правил:**
- 🌬️ Ветер (вправо)
- ⚡ Молнии
- 🌊 Река (преграда)
- 💧 Влажность

**Параметры:**
- **Влажность:** 0-100% (влияет на вероятность возгорания)
- **Скорость:** 1-10x (скорость анимации)

**Интерактив:**
- Кликните по любому дереву на поле, чтобы поджечь его!

## 4. РЕЗУЛЬТАТЫ РАБОТЫ

### Тест 1: Базовая модель (без правил)
**Условия:**
- Все правила выключены
- Поджог в центре

**Результат:**
- Огонь распространяется равномерно во все стороны
- Форма пожара приближена к кругу
- Скорость распространения: ~3-4 клетки за шаг
- Всё дерево сгорает за 20-30 шагов

**Вывод:** Без дополнительных правил пожар ведёт себя предсказуемо, распространяясь от центра к краям.

---

### Тест 2: Правило "Ветер"
**Условия:**
- Включено: Ветер (вправо)
- Поджог в центре

**Результат:**
- Огонь распространяется быстрее вправо (по ветру)
- Форма пожара вытянута вправо
- Слева от центра огонь затухает быстрее
- Скорость по ветру: ~5-6 клеток за шаг
- Скорость против ветра: ~1-2 клетки за шаг

**Вывод:** Ветер значительно влияет на направление и скорость распространения пожара.

---

### Тест 3: Правило "Река"
**Условия:**
- Включено: Река (преграда посередине)
- Поджог слева от реки

**Результат:**
- Огонь не может пересечь реку
- Левая часть леса выгорает полностью
- Правая часть остаётся зелёной (если нет молний)
- Река (голубая линия) остаётся неизменной

**Вывод:** Преграды эффективно останавливают распространение огня.

---

### Тест 4: Правило "Молнии"
**Условия:**
- Включено: Молнии (0.3%)
- Базовая модель

**Результат:**
- Появляются случайные очаги пожара в разных местах
- Даже без поджога пожар может начаться сам
- Несколько очагов сливаются в один большой пожар
- Общая площадь пожара увеличивается на 30-40%

**Вывод:** Молнии создают дополнительную опасность и непредсказуемость.

---

### Тест 5: Правило "Влажность"
**Условия:**
- Включено: Влажность 80%
- Поджог в центре

**Результат:**
- Огонь распространяется медленнее
- Многие деревья не загораются от соседей
- Пожар часто затухает сам
- Площадь пожара меньше на 50-60%

*При влажности 0%:*
- Огонь распространяется очень быстро
- Выгорает почти весь лес

**Вывод:** Влажность — эффективный фактор сдерживания пожара.

---

### Тест 6: Все правила вместе
**Условия:**
- Включены все 4 правила
- Влажность: 50%
- Поджог в центре

**Результат:**
- Сложная динамика пожара
- Ветер тянет огонь вправо
- Река останавливает с одной стороны
- Молнии создают новые очаги справа от реки
- Влажность замедляет общее распространение

**Вывод:** Комбинация правил создаёт реалистичную модель лесного пожара.
